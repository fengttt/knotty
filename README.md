# knotty

Knotty, knot.  Vibe me, vibe me knot.  Why knot.  

This is an exercise for vibe coding a knot database plus some knot puzzle/games.  
Knotty tries to vibe code the whole thing.  Human should not write or debug a single 
line of code.

Knotty uses data and code from these amazing knot sites, [KnotAtlas](https://katlas.org), 
[KnotInfo](https://knotinfo.org), and [KnotFolio](https://knotfol.io).
